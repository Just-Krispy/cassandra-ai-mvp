# Nash Visualization Library 🦞📊

A modern visualization library for Nash Intelligence and related projects. Includes 10 data-driven chart components:

- Built with [Chart.js](https://www.chartjs.org/) and [D3.js](https://d3js.org/)
- Nash dark mode theme
- Ready for integration in web apps, dashboards, and research tools
- Includes demo, usage docs, and sample data

## Features

- Custom Nash theme and palette
- 10 advanced visualization components (see demo.html)
- Animation, export, tooltip helpers
- Ultra-clean, responsive layouts

## Usage

1. **Clone or download this repo**
2. **Open `demo.html` in your browser** – everything is demo-ready with CDN links for Chart.js & D3.js, no install required
3. Review `viz-library.js` for utility functions
4. Integrate your chosen component into your project. Sample data is provided in `/sample-data/`.
