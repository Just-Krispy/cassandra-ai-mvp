🔮 Cassandra AI Decision Tree Visualization

Interactive Features

  • Click nodes to expand/collapse branches
  • Hover nodes to see detailed tooltips
  • Scroll to zoom, drag to pan
  • Color coding: Blue = decisions, Amber = chance nodes, Green = positive outcomes, Red = negative outcomes

Integration Example

To use this visualization in your application:

// Include D3.js v7
<script src="https://d3js.org/d3.v7.min.js"></script>
<link rel="stylesheet" href="tree-viz.css">
<script src="tree-viz.js"></script>

// Render the tree
const container = document.getElementById('tree-container');
const viz = renderDecisionTree(treeData, container);

// Tree data format:
const treeData = {
  name: "Decision question",
  type: "decision",
  children: [
    {
      name: "Option A",
      type: "chance",
      probability: "60%",
      children: [
        { name: "Good outcome", type: "outcome", value: 100, outcome: "positive" },
        { name: "Bad outcome", type: "outcome", value: -50, outcome: "negative" }
      ]
    }
  ]
};
